✉️ Brevo — Email 行銷與試看名單

負責收集試看名單、自動寄出試看 Email、觸發 EDM 序列。

設定步驟

1
前往 app.brevo.com → 右上角帳號 → SMTP & API → 建立 API Key
2
建立聯絡人清單：Contacts → Lists → Create a list，複製 List ID（數字）
3
驗證寄件人 Email：Senders & IP → Senders → Add a Sender，完成驗證
4
到 Vercel Dashboard → Settings → Environment Variables 填入以下變數：
# Brevo 設定 BREVO_API_KEY=xkeysib-xxx...xxx BREVO_LIST_ID=3 # 替換為你的 List ID BREVO_SENDER_EMAIL=hello@yourdomain.com BREVO_SENDER_NAME=InRecord DEMO_URL=https://yourdomain.com/#courseDemo # 可選：使用 Brevo 模板 # BREVO_TEMPLATE_ID=12

💳 Stripe — 課程金流

負責 Checkout Session 建立、付款處理、Webhook 自動開課。

設定步驟

1
前往 dashboard.stripe.com → Developers → API keys，複製 Secret key（sk_live_xxx 或 sk_test_xxx）
2
前往 Products → Add product，為以下每個方案各建立一個 Price，複製 price_xxx ID：
方案金額環境變數名稱說明
粉絲限定【1】NT$2,200STRIPE_PRICE_FAN16.9 折
粉絲限定【2】NT$2,400STRIPE_PRICE_FAN27.5 折
第一波早鳥【1】NT$2,800STRIPE_PRICE_EARLY18.1 折
第二波早鳥【2】NT$3,100STRIPE_PRICE_EARLY28.8 折
最後早鳥【3】NT$3,300STRIPE_PRICE_EARLY39.4 折
原價NT$3,500STRIPE_PRICE_FULL無折扣
3
到 Vercel 填入環境變數：
# Stripe 設定 STRIPE_SECRET_KEY=sk_live_xxx # 或 sk_test_xxx 先測試 STRIPE_PRICE_FAN1=price_xxx STRIPE_PRICE_FAN2=price_xxx STRIPE_PRICE_EARLY1=price_xxx STRIPE_PRICE_EARLY2=price_xxx STRIPE_PRICE_EARLY3=price_xxx STRIPE_PRICE_FULL=price_xxx STRIPE_SUCCESS_URL=https://yourdomain.com/success STRIPE_CANCEL_URL=https://yourdomain.com/#pricing
4
設定 Webhook：Developers → Webhooks → Add endpoint → 填入 https://yourdomain.com/api/stripe/webhook，選取 checkout.session.completed 事件
5
部署後點擊「測試 Stripe 連線」確認環境變數正確
